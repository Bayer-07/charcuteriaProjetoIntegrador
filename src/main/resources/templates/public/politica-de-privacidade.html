Política de Privacidade

Data de Vigência: 8 de Junho de 2026

1. Coleta de Dados

Para processar os pedidos de nossa charcutaria, coletamos os seguintes dados estruturados:

  • Identidade e Roteamento: Nome completo, endereço de cobrança e endereço de entrega.
  • Comunicação: E-mail e telefone de contato.
  • Financeiro: O processamento de pagamentos é gerenciado via APIs externas. Não mantemos persistência de dados de cartão de crédito em nossa infraestrutura.

2. Utilização Operacional

Os dados coletados são estritamente utilizados para:

  • Logística e entrega de produtos perecíveis.
  • Rastreamento de estado transacional e notificações do sistema.
  • Autenticação de usuários e controle de sessão.

3. Segurança e Retenção

Os dados em trânsito são protegidos por criptografia padrão. Os registros de log e do usuário são retidos no banco de dados apenas pelo tempo necessário para auditoria padrão e conclusão do pedido.

4. Direitos do Usuário

Os usuários têm o direito de solicitar a exclusão permanente de sua conta e a limpeza de seus registros de nossos ambientes de armazenamento persistente a qualquer momento.

⚠️ AVISO ACADÊMICO: Esta política de privacidade não possui validade legal. É uma representação de interface gerada exclusivamente para fins de projeto da faculdade.
