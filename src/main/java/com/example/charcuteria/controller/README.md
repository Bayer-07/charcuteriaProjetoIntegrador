### 🎮 `controller/`
Camada de exposição que lida com o protocolo HTTP.
* **`api/`**: Controlladores REST que retornam JSON, utilizados para integrações e consumo via API.
* **`view/`**: Controlladores que gerenciam a navegação do **Thymeleaf**, responsáveis por retornar o nome das páginas HTML a serem renderizadas.
